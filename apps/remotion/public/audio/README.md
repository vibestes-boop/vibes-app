# Audio Assets

Lege hier die Audio-Dateien ab. Danach in `src/lib/audio.ts` das `AVAILABLE_AUDIO`-Set erweitern.

| Dateiname            | Beschreibung                          | Länge   | Genutzt in           |
|----------------------|---------------------------------------|---------|----------------------|
| coin-tick.mp3        | Kurzes Pling für Coin-Counter-Ticks   | ~100ms  | WeeklyTopGifters     |
| coin-collect.mp3     | Satter Coin-Sound bei Gift-Eingang    | ~500ms  | WeeklyTopGifters     |
| intro-jingle.mp3     | Opener für den Live-Stream            | ~1s     | LiveStreamIntro      |
| leaderboard-bg.mp3   | Ambient-Loop für das Leaderboard      | 30s+    | WeeklyTopGifters     |
| fanfare.mp3          | Fanfare bei Rank-1-Reveal             | ~1.5s   | WeeklyTopGifters     |

## Empfohlene Quellen

- https://freesound.org (CC0-Lizenzen)
- https://pixabay.com/music/ (Royalty-free)
- https://mixkit.co/free-sound-effects/

## Format

MP3 (128kbps für Sounds, 320kbps für Musik) oder WAV.
